# 🧘 Habit Tracker & Productivity Analyzer (SQL Project)

This SQL-only project helps track daily habits like reading, exercising, etc., and analyzes productivity patterns using queries.

## 📂 Features
- Track habits by category (Health, Work, Study)
- Store daily progress (done or missed)
- Analyze:
  - Most consistent habits
  - Most missed habits
  - Weekly performance summary

## 🛠️ Technologies
- SQLite 3 (can also work in MySQL/PostgreSQL with tweaks)
- DB Fiddle / Local SQL editors

## 📁 Files
- `schema.sql` → SQL tables (users, habits, categories, entries)
- `queries.sql` → Data + analysis queries
